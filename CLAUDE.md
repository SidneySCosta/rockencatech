Follow the Spec-Driven Development (SDD) workflow from Tech Leads Club.
Reference files are in .specs/ and .claude/ (if generated by the skill).

## Session Start
Always load:
- .specs/project/STATE.md (persistent memory)

## Workflow
Use the tlc-spec-driven methodology:
- Trigger "initialize project" to start a new project
- Auto-size depth by complexity (Small / Medium / Large / Complex)
- Create atomic tasks with verification criteria
- Commit atomically with traceability
